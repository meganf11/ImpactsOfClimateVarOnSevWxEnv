This repo has code from my MS thesis completed in 2022 at the Department of Atmospheric Science at Colorado State University (CSU). 

It is entitled, "Impacts of Forced and Internal Climate Variability on Convective Environments Over the Southeastern United States." 

The primary data source used in this project is from the Community Earth System Model version 2 Large Ensemble (CESM2-LENS) developed at the National Center for Atmospheric Research (NCAR) in Boulder, CO.

This work has since been published in Frontiers in Climate: https://doi.org/10.3389/fclim.2024.1385527

This project was done in collaboration with others at CSU: Jim Hurrell, Kristen Rasmussen, and Lantao Sun.
